# منصة ثواني — Landing Page

موقع تعريفي تفاعلي لمنصة ثواني (بوت واتساب + لوحة إدارة للمتاجر).

## محتوى الموقع
- **Hero**: عرض المنصة + إحصائيات
- **جولة تفاعلية في صفحة الادمن**: 9 أقسام رئيسية، يشرحها بوت عائم
- **تجربة حية**: منيو تجريبي + محاكي محادثة واتساب + لوحة ادمن
- **الدعم الفني**: شرح "فكاهي" لسرعة الاستجابة
- **الباقات**: 3 خطط أسعار
- **CTA نهائي**: البوت يتحدث عن كيف ستتحسن حياة المتجر

## التشغيل المحلي
افتح `index.html` في المتصفح مباشرة — لا يحتاج build، يستخدم Tailwind CDN.

## النشر على GitHub Pages
بعد push على main:
1. اذهب إلى Settings → Pages
2. اختر `Branch: main / root`
3. الموقع سيكون متاحاً على `https://61465.github.io/newsaed/`

## التقنيات
- HTML/CSS/JS vanilla (ملف واحد)
- Tailwind CSS عبر CDN
- خط Cairo (Google Fonts)
- Font Awesome 6
- بدون build، بدون dependencies
